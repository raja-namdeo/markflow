About MarkFlow
-------------

MarkFlow is a modern Markdown to PDF converter with both GUI and CLI interfaces.

----------

* `GitHub Repository <https://github.com/raja-namdeo/markflow>`_
* `PyPI Package <https://pypi.org/project/markflow/>`_

Installation
-----------

Basic installation (CLI only):

.. code-block:: bash

   pip install markflow

Full installation (with GUI):

.. code-block:: bash

   pip install markflow[gui]
