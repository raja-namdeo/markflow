# Welcome to Markdown to PDF Converter

Start typing your Markdown content here. The preview will update automatically.

## Features
- Real-time preview
- File operations (New, Open, Save)
- PDF export
- Syntax highlighting
`raja1`   


## Keyboard Shortcuts
- **Ctrl+N**: New file
- **Ctrl+O**: Open file
- **Ctrl+S**: Save file
- **Ctrl+E**: Export PDF
- **Ctrl+Q**: Quit
